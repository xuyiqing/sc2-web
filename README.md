# sc2-web — SC² events front-end (public test)

Public test deploy of the SC² subscription front end. Source of truth: the private
`sc2-calendar` repo (`web/`). Serves `events.sample.json` (synthetic demo data); switch
to live data with `?feed=live` once the calendar pipeline is populated.

Live: https://xuyiqing.github.io/sc2-web/
